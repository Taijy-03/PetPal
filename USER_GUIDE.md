## 1. Getting Started

### First Launch

When you open PetPal for the first time, you'll see the **Dashboard** with an empty state. The app will prompt you to add your first pet. Tap the **"Add Your First Pet"** button to get started!

### Navigation

PetPal uses a **bottom tab bar** with 4 tabs:

| Tab | Icon | Description |
|-----|------|-------------|
| **Home** | 🏠 | Dashboard with overview & stats |
| **Pets** | 🐾 | Full list of all your pets |
| **Reminders** | ⏰ | All reminders across pets |
| **Settings** | ⚙️ | App preferences & data |

## 2. Dashboard (Home)

The Dashboard is your central hub. Here's what you'll find:

### Stats Cards
A horizontal scrolling row at the top showing:
- 🐾 **Total Pets** — Number of pets you've added
- 🏥 **Health Records** — Total health records across all pets
- 🏃 **Activities** — Total logged activities
- ⏰ **Reminders** — Number of active reminders

### Your Pets
A list of all your pets with their photo, name, breed, and age. Tap any pet to view their full profile.

### Recent Activity
Shows the last 5 activities you've logged (e.g., "Walked Buddy — 30 min").

### Upcoming Reminders
Displays the next 3 upcoming active reminders so you never miss anything important.

### Pull to Refresh
Swipe down on the dashboard to refresh all data.

## 3. Managing Pets

### Adding a Pet

1. Tap the **➕ button** (floating action button) on the Home or Pets screen
2. Fill in the pet profile:

| Field | Required | Description |
|-------|----------|-------------|
| **Photo** | No | Tap the camera icon to take a photo or choose from gallery |
| **Name** | ✅ Yes | Your pet's name |
| **Type** | No | Select from: Dog, Cat, Bird, Fish, Rabbit, Hamster, Turtle, Snake, Other |
| **Breed** | No | Breed or species |
| **Gender** | No | Male, Female, or Unknown |
| **Birth Date** | No | Format: YYYY-MM-DD (e.g., 2020-06-15) |
| **Weight** | No | Numeric value with unit (kg or lbs) |
| **Color** | No | Color or markings description |
| **Microchip ID** | No | Microchip identification number |
| **Notes** | No | Any additional notes |

3. Tap **"Save Pet"** to add the pet

### Editing a Pet

- From the **Pet Detail** screen, tap the **✏️ edit** button in the header
- Or from the **Pets List**, **long-press** on a pet and select **"Edit"**

### Deleting a Pet

- From the **Pet Detail** screen, tap the **🗑️ delete** button
- Or from the **Pets List**, **long-press** on a pet and select **"Delete"**
- ⚠️ **Warning**: Deleting a pet will also delete all its health records, activities, and reminders

## 4. Pet Details

Tap on any pet to view their detailed profile. The Pet Detail screen has **4 tabs**:

### 📋 Info Tab
- Full profile details (type, breed, gender, age, weight, color, microchip, notes)
- **Quick Actions** — Buttons to quickly add a health record, log an activity, set a reminder, or add a photo

### 🏥 Health Tab
- Chronological list of all health records
- Each record shows: type badge, title, date, vet name, and notes
- Tap **"Add Record"** to create a new health record

### 🏃 Activity Tab
- Log of all activities for this pet
- Shows activity type icon, duration, distance, and relative timestamp (e.g., "2 days ago")

### 📸 Gallery Tab
- Photo grid of all pet photos
- Tap **"Add Photo"** to select images from your photo library

## 5. Health Records

Health records help you keep track of your pet's medical history.

### Adding a Health Record

1. Navigate to a pet's detail screen → **Health** tab → **"Add Record"**

2. Fill in the form:

| Field | Required | Description |
|-------|----------|-------------|
| **Type** | Yes | Select one: Vaccination, Checkup, Medication, Surgery, Dental, Allergy, Injury, Weight Check, Note |
| **Title** | ✅ Yes | Brief description (e.g., "Annual Rabies Vaccination") |
| **Date** | No | Date of the record (YYYY-MM-DD) |

| **Clinic** | No | Clinic name |
| **Cost** | No | Cost of the visit/treatment |
| **Next Due Date** | No | When the next follow-up is needed |

## 7. Reminders

Never forget an important pet task with PetPal's reminder system.

### Creating a Reminder

1. Navigate to a pet's detail screen → use the **Quick Action** "Reminder" button
2. Fill in the form:

| Field | Required | Description |
|-------|----------|-------------|
| **Title** | ✅ Yes | What the reminder is for (e.g., "Feed Max") |
| **Type** | Yes | Feeding, Walk, Medicine, Vet, Grooming, Vaccination, Other |
| **Date** | No | When the reminder should trigger (YYYY-MM-DD) |
| **Time** | No | Time of the reminder (HH:MM) |
| **Frequency** | No | Once, Daily, Weekly, Monthly, or Yearly |
| **Notes** | No | Additional details |

3. Tap **"Save Reminder"**

### Managing Reminders

On the **Reminders** tab (bottom navigation):

- **Toggle On/Off**: Use the switch on each reminder to activate or deactivate it
- **Visual Indicators**:
  - ✅ Active reminders appear normally
  - 🔕 Inactive reminders appear dimmed with a strikethrough title
  - ⚠️ Past-due reminders are highlighted with a warning border
- **Delete**: Tap the trash icon or long-press to delete (with confirmation)

### Reminder Frequencies

| Frequency | Behavior |
|-----------|----------|
| **Once** | Triggers one time only |
| **Daily** | Repeats every day |
| **Weekly** | Repeats every week |
| **Monthly** | Repeats every month |
| **Yearly** | Repeats every year |

## 10. Tips & Tricks

### 💡 Quick Tips

1. **Long-press on pets** in the Pets List for quick edit/delete actions
2. **Pull down to refresh** on any list screen to reload data
3. **Use Quick Actions** on the Pet Detail Info tab to quickly add records without navigating away
4. **Set recurring reminders** (daily/weekly) for routine tasks like feeding or walks
5. **Add a pet photo** — it makes your pet list much more fun and easier to navigate!

### 📱 Best Practices

- **Keep health records updated** — Log vaccinations and vet visits as they happen
- **Use the notes field** — Add extra context to any record for future reference
- **Set up reminders early** — Don't rely on memory for important pet care tasks
- **Add microchip IDs** — Having this in your phone can be crucial if your pet gets lost
- **Regular activity logging** — Helps you track your pet's exercise and routine patterns

### 🔒 Data Storage

All your data is stored **locally on your device** using AsyncStorage. This means:
- ✅ Your data is private and never sent to any server
- ✅ Data persists between app sessions
- ⚠️ If you uninstall the app, your data will be lost
- ⚠️ Data does not sync between devices
- 💡 Use the **Export Data** feature in Settings to keep a backup

## ❓ FAQ

### Q: Is my data stored online?
**A:** No. All data is stored locally on your device. It's completely private.

### Q: Can I use PetPal on multiple devices?
**A:** Currently, data is stored locally and does not sync across devices. Cloud sync may be added in a future update.

### Q: What happens if I delete the app?
**A:** All data will be lost. Consider using the Export Data feature before uninstalling.

### Q: How many pets can I add?
**A:** There's no limit! Add as many pets as you'd like.

### Q: Does PetPal send push notifications?
**A:** The app supports notifications for reminders. Make sure notifications are enabled in both the app Settings and your device settings.

### Q: Is PetPal free?
**A:** Yes! PetPal is completely free to use.
